## Featured Work

### Pain Intensity Prediction from Physiological Signals

Reproducible research pipeline for classifying pain intensity from ECG, EDA/GSR,
and EMG biosignals, based on a BioVid Heat Pain Database capstone study.

- Built a public-safe synthetic demo for an access-controlled biosignal dataset.
- Implemented signal preprocessing, feature extraction, feature ranking, and
  conventional ML model benchmarking.
- Preserved source-aligned capstone artefacts including the 27-feature
  backward-selected result and reported model metrics.
- Licensed for non-commercial research and portfolio use.

Repository: [pain-intensity-prediction](https://github.com/xavierx88/pain-intensity-prediction)
